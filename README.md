## Getting Setup

> #### Clone Repository
>  ```https://github.com/neel376/CareDash-site.git```

> #### Enter the directory
>  ```cd CareDash-site```


> #### Configure db.yaml file
> 	* run ```npm install```
> 	* run ```npm run dev``` <br/>
> 	* Open up on http://localhost:3000 <br/>
